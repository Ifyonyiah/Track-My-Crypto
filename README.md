# Cryptocurrency Tracker


## Built With:

- React JS
- Material UI 
- Chart JS
- Firebase

### Description:
 
Pulls data from the CoinGecko Data Market API and displays the top 5 countries using cryptocurrencies:

1. USA
2. India
3. Japan
4. Nigeria
5. Ukraine

### Live Demo:

Deployed on Netlify: https://track-mycrypto.netlify.app/
